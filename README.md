# Pulse Energy (MERN)

Full-stack MERN web app for **Pulse Energy**.

## Tech

- MongoDB + Mongoose
- Express.js + Node.js
- React (Vite)
- Tailwind CSS
- JWT authentication

## Project structure

```
/root
  /client
  /server
```

## Environment variables

Create these files (see examples):

- `server/.env` from `server/.env.example`
- `client/.env` from `client/.env.example`

## Run instructions

### Backend

```bash
cd server
npm install
npm run seed
npm run dev
```

### Frontend

```bash
cd client
npm install
npm run dev
```

Open the app at `http://localhost:5173`.
